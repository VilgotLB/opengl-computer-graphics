# OpenGL Computer Graphics

This repository contains OpenGL applications developed for a computer graphics course, including 3D rendering, animation and data visualization.

## Projects

### 1. Solar System Animation

A 3D animation of a solar system built with Python and OpenGL.

Features:
- Animated planets, moons and the sun
- Local and orbital rotation
- Lighting with ambient, diffuse and specular components
- Shader-based rendering
- Object-oriented structure for celestial bodies and camera objects

### 2. Data Visualization

A data visualization application for rendering weighted 2D data points in OpenGL.

Features:
- Visualization of weighted 2D data points
- Coordinate system transformations
- Efficient rendering using minimized draw calls
- Circle rendering using GLSL shaders
- Scaling and coloring based on data attributes

## Technologies

- Python

## Running the Project

Install dependencies:

```bash
pip install PyOpenGL numpy pygame
```

Run one of the applications:

```bash
python task1.py
```

or

```bash
python task2.py
```
